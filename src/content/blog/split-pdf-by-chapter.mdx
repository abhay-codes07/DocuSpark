# Split PDF by chapter for cleaner collaboration

Long PDFs become hard to review when everyone comments in one large file.

## Define logical ranges first

Split by chapter, section, or ownership area so each reviewer gets only relevant content.

## Keep naming consistent

Use predictable names like `chapter-01-intro.pdf` and `chapter-02-method.pdf` for easy tracking.

## Re-merge when finalizing

After edits, merge reviewed sections into one final version for publishing or submission.
