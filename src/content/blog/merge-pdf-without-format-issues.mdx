# How to merge PDF files without losing formatting

Merging PDFs sounds simple, but formatting issues usually happen when pages have mixed sizes, fonts, or scanned layers.

## Start with a clear file order

Rename files with prefixes like `01-`, `02-`, `03-` before uploading. This prevents accidental order shifts when selecting files quickly.

## Use one output style

If your source files mix portrait and landscape pages, keep original orientation but avoid random margin edits during export.

## Final check before sharing

Open the merged file and skim page breaks, headers, and image quality. A 30-second review prevents back-and-forth later.
