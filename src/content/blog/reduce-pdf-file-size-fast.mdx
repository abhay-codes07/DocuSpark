# 5 ways to reduce PDF file size before sharing

Large PDFs slow down collaboration and often fail in upload forms with strict limits.

## 1. Remove duplicate pages

Before compression, delete accidental repeats and blank pages. This gives immediate size savings.

## 2. Compress embedded images

Scanned documents are usually image-heavy. Use medium compression for a good readability-to-size balance.

## 3. Flatten unnecessary layers

Complex design exports may include multiple hidden layers. Flattening can reduce size significantly.

## 4. Avoid re-export loops

Repeated export cycles degrade quality. Keep one source file and generate one optimized output.

## 5. Verify readability

Always test zoom at 100% before sending. Smaller files should still remain easy to read.
